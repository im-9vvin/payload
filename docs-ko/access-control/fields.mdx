필드 접근 제어는 문서 내의 특정 [필드](../fields/overview)에 대한 접근을 제한하는 데 사용되는 [접근 제어](../access-control/overview)입니다.

필드에 접근 제어를 추가하려면 [필드 구성](../fields/overview)에서 `access` 속성을 사용하세요:

```ts
import type { Field } from 'payload'

export const FieldWithAccessControl: Field = {
  // ...
  access: {
    // highlight-line
    // ...
  },
}
```

<Banner type="warning">
  **참고:** 필드 접근 제어는 [컬렉션 접근
  제어](./collections)와 달리 [쿼리](../queries/overview) 제약 조건 반환을
  지원하지 않습니다.
</Banner>

## 구성 옵션

접근 제어는 요청의 작업에 따라 구체적입니다.

필드에 접근 제어를 추가하려면 [필드 구성](../fields/overview)에서 `access` 속성을 사용하세요:

```ts
import type { CollectionConfig } from 'payload';

export const Posts: CollectionConfig = {
  slug: 'posts',
  fields: [
    {
      name: 'title',
      type: 'text',
      // highlight-start
      access: {
        create: ({ req: { user } }) => { ... },
        read: ({ req: { user } }) => { ... },
        update: ({ req: { user } }) => { ... },
      },
      // highlight-end
    };
  ],
};
```

다음 옵션들을 사용할 수 있습니다:

| 함수         | 목적                                                                                              |
| ------------ | ------------------------------------------------------------------------------------------------- |
| **`create`** | 새 문서를 생성할 때 필드 값을 설정할 수 있는 능력을 허용하거나 거부합니다. [자세한 내용](#create). |
| **`read`**   | 필드 값을 읽을 수 있는 능력을 허용하거나 거부합니다. [자세한 내용](#read).                       |
| **`update`** | 필드 값을 업데이트할 수 있는 능력을 허용하거나 거부합니다. [자세한 내용](#update).               |

### Create

새 문서를 생성할 때 필드 값을 설정할 수 있는 능력을 허용하거나 거부하는 boolean을 반환합니다. `false`가 반환되면 전달된 모든 값이 버려집니다.

**사용 가능한 인수 속성:**

| 옵션              | 설명                                                                                                             |
| ----------------- | ---------------------------------------------------------------------------------------------------------------- |
| **`req`**         | 현재 인증된 `user`를 포함하는 [Request](https://developer.mozilla.org/en-US/docs/Web/API/Request) 객체          |
| **`data`**        | 문서를 생성하기 위해 전달된 전체 데이터입니다.                                                                   |
| **`siblingData`** | 문서를 생성하기 위해 전달된 바로 인접한 필드 데이터입니다.                                                       |

### Read

필드 값을 읽을 수 있는 능력을 허용하거나 거부하는 boolean을 반환합니다. `false`인 경우 전체 속성이 결과 문서에서 생략됩니다.

**사용 가능한 인수 속성:**

| 옵션              | 설명                                                                                                             |
| ----------------- | ---------------------------------------------------------------------------------------------------------------- |
| **`req`**         | 현재 인증된 `user`를 포함하는 [Request](https://developer.mozilla.org/en-US/docs/Web/API/Request) 객체          |
| **`id`**          | 읽고 있는 문서의 `id`                                                                                            |
| **`doc`**         | 전체 문서 데이터입니다.                                                                                          |
| **`siblingData`** | 읽고 있는 문서의 바로 인접한 필드 데이터입니다.                                                                   |

### Update

필드 값을 업데이트할 수 있는 능력을 허용하거나 거부하는 boolean을 반환합니다. `false`가 반환되면 전달된 모든 값이 버려집니다.

`false`가 반환되고 필드 값을 업데이트하려고 시도하면 작업에서 오류를 발생시키지 **않지만** 필드는 업데이트 작업에서 생략되고 값은 변경되지 않습니다.

**사용 가능한 인수 속성:**

| 옵션              | 설명                                                                                                             |
| ----------------- | ---------------------------------------------------------------------------------------------------------------- |
| **`req`**         | 현재 인증된 `user`를 포함하는 [Request](https://developer.mozilla.org/en-US/docs/Web/API/Request) 객체          |
| **`id`**          | 업데이트되고 있는 문서의 `id`                                                                                    |
| **`data`**        | 문서를 업데이트하기 위해 전달된 전체 데이터입니다.                                                               |
| **`siblingData`** | 문서를 업데이트하기 위해 전달된 바로 인접한 필드 데이터입니다.                                                   |
| **`doc`**         | 업데이트가 적용되기 전의 전체 문서 데이터입니다.                                                                 |
