Payload는 작업하는 내용에 관계없이 프로젝트를 시작하는 데 도움이 되는 방대한 예제 배열을 제공합니다. 이러한 예제는 쉽게 시작하고 실행할 수 있도록, 그리고 이해하기 쉽도록 설계되었습니다. 시연되는 특정 기능만을 보여주므로 정확히 무엇이 일어나고 있는지 쉽게 파악할 수 있습니다.

- [인증](https://github.com/payloadcms/payload/tree/main/examples/auth)
- [커스텀 컴포넌트](https://github.com/payloadcms/payload/tree/main/examples/custom-components)
- [초안 프리뷰](https://github.com/payloadcms/payload/tree/main/examples/draft-preview)
- [이메일](https://github.com/payloadcms/payload/tree/main/examples/email)
- [폼 빌더](https://github.com/payloadcms/payload/tree/main/examples/form-builder)
- [라이브 프리뷰](https://github.com/payloadcms/payload/tree/main/examples/live-preview)
- [멀티 테넌트](https://github.com/payloadcms/payload/tree/main/examples/multi-tenant)
- [Tailwind / Shadcn-ui](https://github.com/payloadcms/payload/tree/main/examples/tailwind-shadcn-ui)
- [화이트 라벨 관리자 UI](https://github.com/payloadcms/payload/tree/main/examples/whitelabel)

예제를 실행하고 싶다면 `create-payload-app`을 사용하여 하나에서 프로젝트를 생성할 수 있습니다:

```sh
npx create-payload-app --example example_name
```

매일 새로운 예제를 추가하고 있으므로, 특정 사용 사례가 기존 예제에서 시연되지 않는다면 새로운 [토론](https://github.com/payloadcms/payload/discussions)을 시작하거나 새로운 [PR](https://github.com/payloadcms/payload/pulls)을 열어 직접 추가해 주시기 바랍니다.
