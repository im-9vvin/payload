다음은 Payload가 제공하는 모든 Rich Text Features입니다. 모든 것은 커스터마이징 가능하며, [자신만의 features를 생성](/docs/rich-text/custom-features)하거나 기존 features를 수정하여 커뮤니티와 공유할 수 있습니다.

## Features 개요

| Feature 이름                    | 기본 포함 | 설명                                                                                                                                                                         |
| ------------------------------- | ------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`BoldFeature`**               | Yes                 | 굵은 글씨 서식 지원을 추가합니다.                                                                                                                                              |
| **`ItalicFeature`**             | Yes                 | 기울임꼴 텍스트 서식 지원을 추가합니다.                                                                                                                                            |
| **`UnderlineFeature`**          | Yes                 | 밑줄 텍스트 서식 지원을 추가합니다.                                                                                                                                        |
| **`StrikethroughFeature`**      | Yes                 | 취소선 텍스트 서식 지원을 추가합니다.                                                                                                                                     |
| **`SubscriptFeature`**          | Yes                 | 아래첨자 텍스트 서식 지원을 추가합니다.                                                                                                                                         |
| **`SuperscriptFeature`**        | Yes                 | 위첨자 텍스트 서식 지원을 추가합니다.                                                                                                                                       |
| **`InlineCodeFeature`**         | Yes                 | 인라인 코드 서식 지원을 추가합니다.                                                                                                                                            |
| **`ParagraphFeature`**          | Yes                 | 명시적 단락 생성 또는 변환을 위해 슬래시 메뉴와 툴바 드롭다운 모두에 항목을 제공합니다.                                                                         |
| **`HeadingFeature`**            | Yes                 | 제목 노드를 추가합니다 (기본적으로 H1 - H6, 하지만 커스터마이징 가능)                                                                                                                |
| **`AlignFeature`**              | Yes                 | 텍스트 정렬 지원을 추가합니다 (왼쪽, 가운데, 오른쪽, 양쪽 맞춤)                                                                                                                      |
| **`IndentFeature`**             | Yes                 | 툴바 버튼으로 텍스트 들여쓰기 지원을 추가합니다                                                                                                                              |
| **`UnorderedListFeature`**      | Yes                 | 순서 없는 목록(ul) 지원을 추가합니다                                                                                                                                               |
| **`OrderedListFeature`**        | Yes                 | 순서 있는 목록(ol) 지원을 추가합니다                                                                                                                                                 |
| **`ChecklistFeature`**          | Yes                 | 대화형 체크리스트 지원을 추가합니다                                                                                                                                             |
| **`LinkFeature`**               | Yes                 | 내부 및 외부 링크 생성을 허용합니다                                                                                                                                    |
| **`RelationshipFeature`**       | Yes                 | 다른 문서에 대한 블록 레벨 (인라인이 아닌) 관계를 생성할 수 있습니다                                                                                                      |
| **`BlockquoteFeature`**         | Yes                 | 블록 레벨 인용문 생성을 허용합니다                                                                                                                             |
| **`UploadFeature`**             | Yes                 | 블록 레벨 업로드 노드 생성을 허용합니다 - 이미지뿐만 아니라 모든 종류의 업로드를 지원합니다                                                                                 |
| **`HorizontalRuleFeature`**     | Yes                 | 수평선 / 구분선 지원을 추가합니다. 기본적으로 `<hr>` 요소를 표시합니다                                                                                                |
| **`InlineToolbarFeature`**      | Yes                 | 텍스트를 선택할 때 나타나는 플로팅 툴바를 제공합니다. 이 툴바는 선택된 텍스트와 관련된 작업만 포함합니다                                                       |
| **`FixedToolbarFeature`**       | No                  | 상단에 고정되어 항상 표시되는 지속적인 툴바를 제공합니다. 인라인과 고정 툴바는 동시에 활성화할 수 있습니다.                                                 |
| **`BlocksFeature`**             | No                  | Payload의 [Blocks Field](../fields/blocks)를 에디터 내부에서 직접 사용할 수 있습니다. feature props에서 Blocks field처럼 허용된 블록을 지정할 수 있습니다. |
| **`TreeViewFeature`**           | No                  | 에디터 아래에 디버그 박스를 제공하여 현재 에디터 상태, DOM, 시간 여행을 실시간으로 볼 수 있습니다. 디버깅에 매우 유용합니다                            |
| **`EXPERIMENTAL_TableFeature`** | No                  | 테이블 지원을 추가합니다. 이 feature는 향후 제거되거나 안정적인 lexical 릴리스 내에서도 주요 릴리스 없이 파괴적 변경사항을 받을 수 있습니다.             |
| **`TextStateFeature`**          | No                  | TextNode 내에서 키-값 속성을 저장하고 인라인 스타일을 할당할 수 있습니다.                                                                                            |

## 자세한 설명

### BoldFeature

- 설명: 굵은 글씨 텍스트 서식 지원을 추가하고, 고정 및 인라인 툴바 모두에 적용 버튼을 제공합니다.
- 기본 포함: Yes
- 마크다운 지원: `**굵은글씨**` 또는 `__굵은글씨__`
- 키보드 단축키: Ctrl/Cmd + B

### ItalicFeature

- 설명: 기울임꼴 텍스트 서식 지원을 추가하고, 고정 및 인라인 툴바 모두에 적용 버튼을 제공합니다.
- 기본 포함: Yes
- 마크다운 지원: `*기울임꼴*` 또는 `_기울임꼴_`
- 키보드 단축키: Ctrl/Cmd + I

### UnderlineFeature

- 설명: 밑줄 텍스트 서식 지원을 추가하고, 고정 및 인라인 툴바 모두에 적용 버튼을 제공합니다.
- 기본 포함: Yes
- 키보드 단축키: Ctrl/Cmd + U

### StrikethroughFeature

- 설명: 취소선 텍스트 서식 지원을 추가하고, 고정 및 인라인 툴바 모두에 적용 버튼을 제공합니다.
- 기본 포함: Yes
- 마크다운 지원: `~~취소선~~`

### SubscriptFeature

- 설명: 아래첨자 텍스트 서식 지원을 추가하고, 고정 및 인라인 툴바 모두에 적용 버튼을 제공합니다.
- 기본 포함: Yes

### SuperscriptFeature

- 설명: 위첨자 텍스트 서식 지원을 추가하고, 고정 및 인라인 툴바 모두에 적용 버튼을 제공합니다.
- 기본 포함: Yes

### InlineCodeFeature

- 설명: 고유한 스타일링과 함께 인라인 코드 서식 지원을 추가하고, 고정 및 인라인 툴바 모두에 적용 버튼을 제공합니다.
- 기본 포함: Yes
- 마크다운 지원: \`코드\`

### ParagraphFeature

- 설명: 명시적 단락 생성 또는 변환을 위해 슬래시 메뉴와 툴바 드롭다운 모두에 항목을 제공합니다.
- 기본 포함: Yes

### HeadingFeature

- 설명: 각 활성화된 제목 크기에 대한 툴바 드롭다운과 슬래시 메뉴 항목과 함께 제목 노드(H1-H6) 지원을 추가합니다.
- 기본 포함: Yes
- 마크다운 지원: 줄 시작에서 `#`, `##`, `###`, ...
- 타입:

```ts
type HeadingFeatureProps = {
  enabledHeadingSizes?: HeadingTagType[] // ['h1', 'h2', 'h3', 'h4', 'h5', 'h6']
}
```

- 사용 예시:

```ts
HeadingFeature({
  enabledHeadingSizes: ['h1', 'h2', 'h3'], // 기본값: ['h1', 'h2', 'h3', 'h4', 'h5', 'h6']
})
```

### AlignFeature

- 설명: 텍스트 정렬(왼쪽, 가운데, 오른쪽, 양쪽 맞춤)을 허용하고, 고정 및 인라인 툴바 모두에 적용 버튼을 제공합니다.
- 기본 포함: Yes
- 키보드 단축키: Ctrl/Cmd + Shift + L/E/R/J (왼쪽/가운데/오른쪽/양쪽 맞춤)

### IndentFeature

- 설명: 텍스트 들여쓰기 지원을 추가하고, 고정 및 인라인 툴바 모두에 적용 버튼을 제공합니다.
- 기본 포함: Yes
- 키보드 단축키: Tab (증가), Shift + Tab (감소)
- 타입:

```ts
type IndentFeatureProps = {
  /**
   * 들여쓰기되지 않아야 하는 노드들. 들여쓰기하고 싶지 않은
   * 노드의 "type" 속성입니다.
   * 이것들은: "paragraph", "heading", "listitem",
   * "quote" 또는 다른 들여쓰기 가능한 노드들일 수 있습니다.
   */
  disabledNodes?: string[]
  /**
   * true인 경우, 단락이나 제목과 같은 블록 중간에서
   * Tab을 누르면 tabNode를 삽입하지 않습니다.
   * 대신, Tab은 블록 레벨 들여쓰기에만 사용됩니다.
   * @default false
   */
  disableTabNode?: boolean
}
```

- 사용 예시:

```ts
// 블록 레벨 들여쓰기만 허용
IndentFeature({
  disableTabNode: true,
})
```### UnorderedListFeature

- 설명: 툴바 드롭다운과 슬래시 메뉴 항목과 함께 순서 없는 목록(글머리 기호) 지원을 추가합니다.
- 기본 포함: Yes
- 마크다운 지원: 줄 시작에서 `-`, `*`, 또는 `+`

### OrderedListFeature

- 설명: 툴바 드롭다운과 슬래시 메뉴 항목과 함께 순서 있는 목록(번호 목록) 지원을 추가합니다.
- 기본 포함: Yes
- 마크다운 지원: 줄 시작에서 `1.`

### ChecklistFeature

- 설명: 툴바 드롭다운과 슬래시 메뉴 항목과 함께 대화형 체크리스트 지원을 추가합니다.
- 기본 포함: Yes
- 마크다운 지원: `- [ ]` (체크 안 됨) 또는 `- [x]` (체크됨)

### LinkFeature

- 설명: 툴바 버튼과 자동 URL 변환을 통해 내부 및 외부 링크 생성을 허용합니다.
- 기본 포함: Yes
- 마크다운 지원: `[앵커](url)`
- 타입:

```ts
type LinkFeatureServerProps = {
  /**
   * 에디터에 입력하거나 붙여넣은 URL로부터
   * 링크의 자동 생성을 비활성화합니다,
   * @default false
   */
  disableAutoLinks?: 'creationOnly' | true
  /**
   * 링크 feature에 대한 추가 필드를 정의하는
   * 함수 또는 배열입니다.
   * 이들은 링크 에디터 드로어에 표시됩니다.
   */
  fields?:
    | ((args: {
        config: SanitizedConfig
        defaultFields: FieldAffectingData[]
      }) => (Field | FieldAffectingData)[])
    | Field[]
  /**
   * 필드에 도달했을 때 남은 depth에 관계없이
   * 링크의 내부 doc 기본 필드에 대한
   * 최대 population depth를 설정합니다.
   */
  maxDepth?: number
} & ExclusiveLinkCollectionsProps

type ExclusiveLinkCollectionsProps =
  | {
      disabledCollections?: CollectionSlug[]
      enabledCollections?: never
    }
  | {
      disabledCollections?: never
      enabledCollections?: CollectionSlug[]
    }
```

- 사용 예시:

```ts
LinkFeature({
  fields: ({ defaultFields }) => [
    ...defaultFields,
    {
      name: 'rel',
      type: 'select',
      options: ['noopener', 'noreferrer', 'nofollow'],
    },
  ],
  enabledCollections: ['pages', 'posts'], // 내부 링크용 컬렉션
  maxDepth: 2, // 내부 링크에 대한 Population depth
  disableAutoLinks: false, // URL 자동 변환 허용
})
```

### RelationshipFeature

- 설명: 툴바 버튼과 슬래시 메뉴 항목을 통해 다른 문서에 대한 블록 레벨 관계 생성을 허용합니다.
- 기본 포함: Yes
- 타입:

```ts
type RelationshipFeatureProps = {
  /**
   * 해당 필드에 도달했을 때 남은 depth에 관계없이
   * 이 관계에 대한 최대 population depth를 설정합니다.
   */
  maxDepth?: number
} & ExclusiveRelationshipFeatureProps

type ExclusiveRelationshipFeatureProps =
  | {
      disabledCollections?: CollectionSlug[]
      enabledCollections?: never
    }
  | {
      disabledCollections?: never
      enabledCollections?: CollectionSlug[]
    }
```

- 사용 예시:

```ts
RelationshipFeature({
  disabledCollections: ['users'], // 제외할 컬렉션
  maxDepth: 2, // 관계에 대한 Population depth
})
```

### UploadFeature

- 설명: 툴바 버튼과 슬래시 메뉴 항목을 통해 업로드/미디어 노드 생성을 허용하며, 모든 파일 타입을 지원합니다.
- 기본 포함: Yes
- 타입:

```ts
type UploadFeatureProps = {
  collections?: {
    [collection: CollectionSlug]: {
      fields: Field[]
    }
  }
  /**
   * 해당 필드에 도달했을 때 남은 depth에 관계없이
   * 이 업로드에 대한 (이 업로드의 필드가 아닌)
   * 최대 population depth를 설정합니다.
   */
  maxDepth?: number
}
```

- 사용 예시:

```ts
UploadFeature({
  collections: {
    uploads: {
      fields: [
        {
          name: 'caption',
          type: 'text',
          label: 'Caption',
        },
        {
          name: 'alt',
          type: 'text',
          label: 'Alt Text',
        },
      ],
    },
  },
  maxDepth: 1, // 업로드에 대한 Population depth
})
```

### BlockquoteFeature

- 설명: 툴바 버튼과 슬래시 메뉴 항목을 통해 블록 인용문 생성을 허용합니다.
- 기본 포함: Yes
- 마크다운 지원: `> 인용문 텍스트`

### HorizontalRuleFeature

- 설명: 툴바 버튼과 슬래시 메뉴 항목을 통해 수평선/구분선 지원을 추가합니다.
- 기본 포함: Yes
- 마크다운 지원: `---`

### InlineToolbarFeature

- 설명: 텍스트가 선택될 때 나타나는 플로팅 툴바를 제공하며, 선택된 텍스트와 관련된 서식 옵션을 포함합니다.
- 기본 포함: Yes

### FixedToolbarFeature

- 설명: 에디터 상단에 고정되어 항상 표시되는 지속적인 툴바를 제공합니다.
- 기본 포함: No
- 타입:

```ts
type FixedToolbarFeatureProps = {
  /**
   * @default false
   * 이것이 활성화되면, 툴바는
   * FixedToolbarFeature가 있는 에디터가 아닌
   * 포커스된 에디터에 적용됩니다.
   */
  applyToFocusedEditor?: boolean
  /**
   * 툴바 그룹에 대한 사용자 정의 구성
   * 키는 그룹 키 (예: 'format', 'indent', 'align')
   * 값은 기본 구성과 병합될 부분 ToolbarGroup 객체
   */
  customGroups?: CustomGroups
  /**
   * @default false
   * 고정 툴바가 있는 상위 에디터가 있는 경우,
   * 이 에디터의 툴바를 비활성화합니다.
   */
  disableIfParentHasFixedToolbar?: boolean
}
```

- 사용 예시:

```ts
FixedToolbarFeature({
  applyToFocusedEditor: false, // 포커스된 에디터에 적용
  customGroups: {
    format: {
      // format 그룹에 대한 사용자 정의 구성
    },
  },
})
```

### BlocksFeature

- 설명: 각 블록 타입에 대한 툴바 버튼과 슬래시 메뉴 항목과 함께 Payload의 Blocks Field를 에디터에서 직접 사용할 수 있습니다.
- 기본 포함: No
- 타입:

```ts
type BlocksFeatureProps = {
  blocks?: (Block | BlockSlug)[] | Block[]
  inlineBlocks?: (Block | BlockSlug)[] | Block[]
}
```

- 사용 예시:

```ts
BlocksFeature({
  blocks: [
    {
      slug: 'callout',
      fields: [
        {
          name: 'text',
          type: 'text',
          required: true,
        },
      ],
    },
  ],
  inlineBlocks: [
    {
      slug: 'mention',
      fields: [
        {
          name: 'name',
          type: 'text',
          required: true,
        },
      ],
    },
  ],
})
```

### TreeViewFeature

- 설명: 에디터의 내부 상태, DOM 트리, 시간 여행 디버깅을 보여주는 에디터 아래의 디버그 패널을 제공합니다.
- 기본 포함: No

### EXPERIMENTAL_TableFeature

- 설명: 생성 및 편집을 위한 툴바 버튼과 슬래시 메뉴 항목과 함께 테이블 지원을 추가합니다.
- 기본 포함: No

### TextStateFeature

- 설명: 인라인 스타일과 스타일 선택을 위한 툴바 드롭다운을 통해 텍스트 노드에서 키-값 속성을 저장할 수 있습니다.
- 기본 포함: No
- 타입:

```ts
type TextStateFeatureProps = {
  /**
   * 최상위 객체의 키(stateKeys)는 textNode가 가질 수 있는 속성을 나타냅니다 (예: color).
   * 최상위 객체의 값(stateValues)은 해당 속성이 가질 수 있는 값들을 나타냅니다 (예: red, blue 등).
   * stateValue 내에서 인라인 스타일과 레이블을 정의할 수 있습니다.
   */
  state: { [stateKey: string]: StateValues }
}

type StateValues = {
  [stateValue: string]: {
    css: StyleObject
    label: string
  }
}

type StyleObject = {
  [K in keyof PropertiesHyphenFallback]?:
    | Extract<PropertiesHyphenFallback[K], string>
    | undefined
}
```

- 사용 예시:

```ts
// 다크 모드와 라이트 모드에서 좋은 대비와 보기 좋은 기본 색상들을 제공합니다.
import { defaultColors, TextStateFeature } from '@payloadcms/richtext-lexical'

TextStateFeature({
  // prettier-ignore
  state: {
    color: {
      ...defaultColors,
      // 멋진 그라데이션!
      galaxy: { label: 'Galaxy', css: { background: 'linear-gradient(to right, #0000ff, #ff0000)', color: 'white' } },
      sunset: { label: 'Sunset', css: { background: 'linear-gradient(to top, #ff5f6d, #6a3093)' } },
    },
    // 컬러와 밑줄이 동시에 적용된 텍스트를 가질 수 있습니다.
    // 그것을 원하지 않는다면, 같은 키 내에서 그룹화해야 합니다.
    // (내가 defaultColors와 멋진 그라데이션으로 한 것처럼)
    underline: {
      'solid': { label: 'Solid', css: { 'text-decoration': 'underline', 'text-underline-offset': '4px' } },
       // CSS light-dark() 유틸리티를 사용하는 것을 권장합니다.
      'yellow-dashed': { label: 'Yellow Dashed', css: { 'text-decoration': 'underline dashed', 'text-decoration-color': 'light-dark(#EAB308,yellow)', 'text-underline-offset': '4px' } },
    },
  },
}),
```
