Payload는 TypeScript를 기본적으로 지원하며, 그뿐만 아니라 CMS 전체가 TypeScript로 구축되었습니다. Payload와 TypeScript로 개발을 시작하려면 `create-payload-app`을 통해 한 줄로 Payload의 내장 보일러플레이트 중 하나를 사용할 수 있습니다:

```
npx create-payload-app@latest
```

쉽게 시작할 수 있도록 TypeScript 프로젝트 유형을 선택하세요.

## 처음부터 설정하기

처음부터 TypeScript 프로젝트를 설정하는 것도 가능합니다. 정확히 어떻게 하는지에 대한 가이드를 작성할 계획이므로, 그것도 주목해 주세요.

## Payload의 내보낸 타입 사용하기

Payload는 [플러그인](../plugins/overview), [훅](../hooks/overview), [접근 제어](../access-control/overview) 함수, [커스텀 뷰](../custom-components/custom-views), [GraphQL 쿼리/뮤테이션](../graphql/overview) 또는 기타 모든 것과 같은 자신만의 커스텀 기능을 작성하는 동안 유용할 수 있는 여러 타입을 내보냅니다.
